# Project Blueprint

## Overview
This document outlines the current state, features, and modifications for the MySchol1 Flutter application. The goal is to maintain a clean, stable, and functional application.

## Current Features
*   User authentication (Login, Register)
*   Dashboard for various user roles
*   Academic year management
*   Attendance tracking
*   Book management and library system
*   Class management
*   Event management
*   Grade management
*   Leave management
*   Performance evaluation
*   Permission management
*   Staff and Teacher management
*   Student management and portal
*   Subject management
*   Timetable management
*   User management (Admin)
*   Parent portal
